# 登录界面说明文档

## 界面设计

### 16:9 比例设计
- 整体界面采用 **16:9** 比例设计
- 最大宽度：1280px
- 最大高度：720px
- 响应式设计，适配不同屏幕尺寸

### 背景设计
- 登录界面背景色为**透明**（transparent）
- 设备框架使用半透明背景（rgba(15, 15, 26, 0.85)）
- 添加了毛玻璃效果（backdrop-filter: blur(10px)）
- 边框带有半透明白色边框效果

### 响应式适配
- **大屏幕**（>1366px）：1280x720（16:9）
- **中等屏幕**（1080px-1366px）：1024x576（16:9）
- **小屏幕**（<1080px）：854x480（16:9）
- **移动端**：自动调整高度，保持可用性

### 登录验证
系统已预设默认账号和密码，验证逻辑如下：

```typescript
if (loginForm.value.username === 'admin' && loginForm.value.password === '123456') {
  // 登录成功
  localStorage.setItem('isLoggedIn', 'true');
  localStorage.setItem('username', 'admin');
  router.push('/main');
} else {
  // 登录失败
  alert('用户名或密码错误！\n默认账号: admin\n默认密码: 123456');
}
```

## 使用说明

### 登录流程

1. **打开应用**
   - 访问 `http://localhost:5000`
   - 自动跳转到登录页面

2. **查看账号信息**
   - 登录表单上方显示默认账号提示
   - 账号和密码已预填充

3. **登录系统**
   - 直接点击"登录"按钮
   - 或修改账号密码后点击"登录"

4. **进入主界面**
   - 登录成功后自动跳转到主界面
   - 保存登录状态到 localStorage

### 修改默认账号

如需修改默认账号和密码，编辑 `frontend/src/views/Login.vue`：

```typescript
// 修改这里的默认账号和密码
const DEFAULT_USERNAME = 'admin';  // 修改用户名
const DEFAULT_PASSWORD = '123456'; // 修改密码
```

### 退出登录

在主界面点击右下角退出按钮即可退出登录。

## 技术实现

### 关键技术点

1. **16:9 比例实现**
   ```css
   .device-frame {
     aspect-ratio: 16 / 9;
     width: 100%;
     max-width: 1280px;
     max-height: 720px;
   }
   ```

2. **透明背景**
   ```css
   .login-container {
     background: transparent;
   }
   ```

3. **毛玻璃效果**
   ```css
   .device-frame {
     background: rgba(15, 15, 26, 0.85);
     backdrop-filter: blur(10px);
     -webkit-backdrop-filter: blur(10px);
   }
   ```

4. **预填充表单**
   ```typescript
   const loginForm = ref({
     username: DEFAULT_USERNAME,
     password: DEFAULT_PASSWORD
   });
   ```
