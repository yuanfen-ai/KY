# 手持察打一体设备 - 使用说明

## 项目概述

**项目名称**：手持察打一体设备
**项目版本**：v2.1.0
**项目描述**：反无人机操作控制系统
**设备序列号**：SN20260301001

## 功能特性

### 1. 登录界面
- 用户名/密码登录
- 实时时间显示
- 设备状态显示（网络、电量）
- 自动跳转（已登录用户）
- 响应式设计，适配移动设备

### 2. 主界面
#### 顶部状态栏
- 设备名称显示
- 网络状态（4G/5G）
- 实时时间
- 电量显示

#### 左侧功能导航
- **侦测**：无人机搜索功能
- **干扰**：阻断无人机信号
- **诱骗**：接管无人机控制

#### 中心地图操作区
- 模拟地图显示
- 目标无人机标注
- 点击查看详情
- 地图控制按钮
- 建筑物和地标显示

#### 底部功能状态栏
- 模式切换（侦测/干扰/诱骗）
- 当前模式状态指示

#### 目标信息面板
- 目标ID
- 机型信息
- 经纬度坐标
- 高度、速度信息
- 加入白名单功能

## 技术栈

- **前端框架**：Vue 3.5.28
- **路由管理**：Vue Router 4.6.4
- **状态管理**：Pinia 3.0.4
- **UI组件库**：Element Plus 2.13.2
- **构建工具**：Vite 7.3.1
- **TypeScript**：5.9.3
- **静态服务器**：Node.js原生HTTP服务器

| `/login` | Login.vue | 登录界面 | 否 |
| `/main` | Main.vue | 主界面 | 是 |

## 登录逻辑

### 认证流程
1. 用户输入用户名和密码
2. 点击登录按钮
3. 系统验证凭据（模拟）
4. 保存登录状态到localStorage
5. 跳转到主界面

### 认证状态
- `isLoggedIn`: 存储在localStorage中，值为'true'或'false'
- `username`: 存储用户名

### 退出登录
- 点击右下角退出按钮
- 清除localStorage中的登录状态
- 跳转到登录界面

## 界面说明

### 登录界面特性
- **响应式设计**：适配不同屏幕尺寸
- **设备模拟**：模拟手持设备的外观
- **状态栏**：显示设备实时状态
- **密码可见性切换**：点击眼睛图标显示/隐藏密码
- **加载状态**：登录按钮显示加载状态

### 主界面特性
- **实时时间**：每秒更新一次
- **模式切换**：点击左侧或底部按钮切换工作模式
- **目标交互**：点击无人机图标显示目标信息面板
- **动画效果**：
  - 无人机浮动动画
  - 警戒圈脉冲动画
  - 面板滑入动画
  - 按钮悬停效果

## 注意事项

1. **移动端适配**：界面已针对移动端优化，禁用了缩放功能
2. **localStorage依赖**：登录状态存储在localStorage中，清除浏览器数据会退出登录
3. **路由守卫**：未登录用户无法直接访问主界面
4. **静态文件**：生产环境使用自定义静态服务器，不依赖Vite HMR
