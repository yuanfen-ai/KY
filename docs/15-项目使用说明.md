# 手持察打一体设备 - 使用说明

## 项目概述

**项目名称**：手持察打一体设备
**项目版本**：v2.1.0
**项目描述**：反无人机操作控制系统
**设备序列号**：SN20260301001

## 功能特性

### 1. 登录界面
- **16:9 比例设计**：采用标准的 16:9 宽屏比例
- **透明背景**：登录界面背景色为透明，设备框架半透明
- **毛玻璃效果**：背景模糊，增强视觉层次
- **默认账号**：admin / 密码：123456
- **预填充表单**：自动填充默认账号和密码
- **账号提示**：界面显示默认账号提示信息
- **实时时间显示**：顶部状态栏显示当前时间
- **设备状态显示**：网络状态（4G/5G）、电量显示
- **自动跳转**：已登录用户自动跳转到主界面
- **响应式设计**：适配不同屏幕尺寸

### 2. 主界面
#### 顶部状态栏
- 设备名称显示
- 网络状态（4G/5G）
- 实时时间
- 电量显示

#### 左侧功能导航
- **侦测**：无人机搜索功能
- **干扰**：阻断无人机信号
- **诱骗**：接管无人机控制

#### 中心地图操作区
- 模拟地图显示
- 目标无人机标注
- 点击查看详情
- 地图控制按钮
- 建筑物和地标显示

#### 底部功能状态栏
- 模式切换（侦测/干扰/诱骗）
- 当前模式状态指示

#### 目标信息面板
- 目标ID
- 机型信息
- 经纬度坐标
- 高度、速度信息
- 加入白名单功能

## 登录逻辑

### 默认登录账号
- **用户名**：`admin`
- **密码**：`123456`

### 认证流程
1. 打开登录页面，系统自动预填充默认账号和密码
2. 界面显示默认账号提示："默认账号: admin / 密码: 123456"
3. 点击登录按钮
4. 系统验证凭据（用户名：admin，密码：123456）
5. 验证成功后保存登录状态到localStorage
6. 跳转到主界面

### 认证规则
- 用户名必须为 `admin`
- 密码必须为 `123456`
- 输入错误时会提示正确的账号和密码

### 认证状态
- `isLoggedIn`: 存储在localStorage中，值为'true'或'false'
- `username`: 存储用户名

### 退出登录
- 点击右下角退出按钮
- 清除localStorage中的登录状态
- 跳转到登录界面

## 界面说明

### 登录界面特性
- **16:9 比例设计**：采用标准宽屏比例，最大分辨率 1280x720
- **透明背景**：登录界面背景色为透明
- **毛玻璃效果**：设备框架半透明，背景模糊增强层次感
- **响应式设计**：适配不同屏幕尺寸（1366px、1080px、移动端）
- **设备模拟**：模拟手持设备的外观
- **状态栏**：显示设备实时状态（网络、时间、电量）
- **账号提示**：界面显示默认账号和密码提示
- **预填充表单**：自动填充默认账号和密码
- **密码可见性切换**：点击眼睛图标显示/隐藏密码
- **加载状态**：登录按钮显示加载状态
- **Logo动画**：盾牌图标带有脉冲动画效果

### 主界面特性
- **实时时间**：每秒更新一次
- **模式切换**：点击左侧或底部按钮切换工作模式
- **目标交互**：点击无人机图标显示目标信息面板
- **动画效果**：
  - 无人机浮动动画
  - 警戒圈脉冲动画
  - 面板滑入动画
  - 按钮悬停效果

1. 打开登录页面
2. 查看页面上的默认账号提示
3. 表单已预填充账号和密码
4. 直接点击"登录"按钮即可进入系统

## 注意事项

1. **移动端适配**：界面已针对移动端优化，禁用了缩放功能
2. **localStorage依赖**：登录状态存储在localStorage中，清除浏览器数据会退出登录
3. **路由守卫**：未登录用户无法直接访问主界面
4. **静态文件**：生产环境使用自定义静态服务器，不依赖Vite HMR
