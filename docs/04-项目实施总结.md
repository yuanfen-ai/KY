## 部署准备

### 前端部署

```bash
cd frontend
pnpm install
pnpm build
# 产物在 dist/ 目录
```

### 后端部署

```bash
cd proxy-server
pnpm install
cp .env.example .env
# 编辑 .env 配置数据库等
pnpm build
pm2 start ecosystem.config.js
```

### 数据库初始化

```bash
mysql -u root -p < database/schema.sql
```

## 技术亮点

1. **类型安全**：全栈TypeScript开发，类型定义完整
2. **自动重连**：WebSocket和TCP双向自动重连，指数退避策略
3. **心跳检测**：实时监测连接状态，异常自动断开
4. **模块化设计**：前后端模块清晰，易于维护和扩展
5. **配置管理**：环境变量统一管理，支持多环境部署
6. **日志系统**：结构化日志，支持文件和控制台输出

## 性能指标

| 指标 | 设计目标 | 备注 |
|------|---------|------|
| 并发连接数 | 3000-5000 | 需配置Cluster和负载均衡 |
| 响应延迟 | <1秒 | WebSocket + 异步I/O |
| 重连次数 | 5次 | 指数退避策略 |
| 心跳间隔 | 30秒 | 可配置 |
