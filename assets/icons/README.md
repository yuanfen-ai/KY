# 图标资源文件夹

本文件夹用于存放项目中的图标资源。

## 文件夹结构

```
assets/
├── icons/              # 图标资源（本文件夹）
│   ├── *.png          # PNG格式图标
│   ├── *.svg          # SVG格式图标
│   ├── *.jpg          # JPG格式图标
│   └── ...
├── 1.png              # 其他图片资源
├── image.png
└── 按钮8 拷贝 4.png
```

## 使用说明

1. **图标命名规范**：
   - 使用小写字母和下划线
   - 采用描述性命名（如 `btn_detect_active.png`）
   - 包含用途和状态信息

2. **推荐格式**：
   - SVG：矢量图标，推荐用于需要缩放的图标
   - PNG：位图图标，适用于复杂图形
   - WebP：现代格式，压缩率高

3. **引用方式**：
   ```vue
   <!-- 在Vue组件中引用 -->
   <img src="/assets/icons/your-icon.png" alt="图标描述" />
   ```

4. **尺寸建议**：
   - 小图标：16x16, 24x24, 32x32
   - 中图标：48x48, 64x64
   - 大图标：128x128, 256x256

## 注意事项

- 请保持图标资源的版权合法性
- 优化图标文件大小，避免影响加载性能
- 使用透明背景（PNG/SVG）以适应不同背景色
