## 1. 项目概述

### 1.1 项目背景
手持察打一体设备是一款专为移动端设计的反无人机操作控制系统，采用竖屏布局，提供实时侦测、信号干扰和导航诱骗三大核心功能，适用于无人机黑飞防范、空域管控等场景。

### 1.2 项目目标
- 提供直观的移动端操作界面
- 实现实时无人机侦测与定位
- 支持信号干扰和导航诱骗功能
- 适配手持设备（800×480分辨率）
- 提供安全可靠的权限管理

### 1.3 设计原则
- **移动优先**: 专为手持设备竖屏设计
- **简洁高效**: 界面简洁，操作快捷
- **实时反馈**: 实时显示设备状态和信号数据
- **安全可靠**: 完善的权限控制和状态管理

### 3.3 侦测模式

#### 功能描述
实时显示侦测到的无人机信号和目标信息，支持测向和定位功能。

#### 侦测列表数据结构
```typescript
interface DetectTarget {
  id: number;                              // 目标唯一标识
  type: 'signal' | 'target';               // 类型：信号信息/目标信息
  time?: string;                           // 时间（信号类型）
  signalStrength?: number;                 // 信号强度（0-255）
  frequency?: string;                      // 频率（MHz）
  targetId?: string;                       // 目标ID（目标类型）
  model?: string;                          // 机型（目标类型）
  altitude?: number;                       // 高度（米，目标类型）
  horizontalSpeed?: number;                // 水平速度（米/秒，目标类型）
  verticalSpeed?: number;                  // 垂直速度（米/秒，目标类型）
  lat?: string;                            // 纬度（目标类型）
  lng?: string;                            // 经度（目标类型）
  top?: string;                            // 地图位置top（目标类型）
  left?: string;                           // 地图位置left（目标类型）
  buttonType: 'measure' | 'locate';        // 按钮类型：测向/定位
  buttonActive: boolean;                   // 按钮激活状态
}
```

#### 侦测列表展示
- **白色背景卡片设计**
- **两列布局**：信息列（80%） + 操作按钮列（20%）
- **信号信息显示**：时间、信号强度、频率
- **目标信息显示**：目标ID、机型、高度、水平速度、垂直速度
- **操作按钮**：测向按钮（信号类型）/ 定位按钮（目标类型）
- **按钮状态**：灰色（未激活） → 绿色（激活）

#### 测向功能
- **触发方式**: 点击侦测列表中的"测向"按钮
- **显示位置**: 地图顶部中间
- **UI组件**:
  - 信号图标（左侧）
  - 渐变色进度条（中间）
  - 信号数值（右侧）
- **信号图标**: 信号发射波纹样式
  - 底部中心：信号源圆点（蓝色填充）
  - 三层信号弧：向上扩散的同心圆弧
- **进度条样式**:
  - 渐变：浅蓝 → 浅黄 → 橙色
  - 范围：0-100%（映射信号强度0-255）
- **数据接口**: 预留 `onSignalDataReceived(data)` 方法接收后端实时数据
- **隐藏条件**: 点击定位按钮、切换到干扰/诱骗模式、点击地图空白区域

#### 定位功能
- **触发方式**: 点击侦测列表中的"定位"按钮
- **显示效果**: 地图上对应无人机目标高亮显示
- **信息弹窗**: 右下角悬浮框显示目标详细信息
- **详细信息**: 目标ID、机型、位置（经纬度）、高度、水平速度、垂直速度

## 6. 数据接口

### 6.1 预留接口

#### 信号数据接收接口
```typescript
/**
 * 接收后端信号数据
 * @param data 信号数据
 * @param data.value 信号强度值（0-255）
 */
const onSignalDataReceived = (data: { value: number }) => {
  if (showSignalProgress.value) {
    signalValue.value = data.value;
    updateSignalProgress(data.value);
    console.log('[MainPage] 接收到信号数据:', data.value);
  }
};
```

**使用场景**: WebSocket消息处理、API响应处理

**调用示例**:
```typescript
// WebSocket消息处理
socket.onmessage = (event) => {
  const data = JSON.parse(event.data);
  onSignalDataReceived(data);
};

// API响应处理
axios.get('/api/signal').then(response => {
  onSignalDataReceived(response.data);
});
```

## 7. 安全设计

### 7.1 权限控制
- **路由守卫**: 所有需要登录的页面均通过路由守卫验证
- **会话管理**: 登录状态保存在localStorage
- **自动跳转**: 未登录用户访问受保护页面自动跳转到登录页

### 7.2 输入验证
- **登录表单**: 用户名和密码必填验证
- **数字输入**: 高度、经纬度等数字输入限制
- **范围验证**: 信号强度0-255，经纬度范围验证

### 7.3 防护措施
- **XSS防护**: Vue3自动转义，避免v-html直接使用
- **CSRF防护**: 后端接口需实现CSRF Token（预留）
- **敏感信息**: 密码不在URL中传递，使用HTTPS传输（生产环境）
